# Cryptocurrency trading platform

## 已实现的主要功能：
1. 首页：极具冲击力的英雄部分，运用了玻璃变形、霓虹发光效果和便当网格特色展示。
2. 投资组合仪表盘：实时净值跟踪重新绘制图表面积图、资产配置环形图和详细的交易历史记录。
3. 交易终端：功能型交易界面，包括模拟订单簿、蜡烛图式深度图和双模式（买入/卖出）交易面板。
4. 收益终端：为流动性提供者和质押者提供专门的视图，包括收益计算器、主动质押管理和特色流动性池。

## 设计系统：
1. 字体排印：用途太空怪诞打造高影响力标题和国际为了获得最佳的用户界面清晰度。
2. 配色方案：深炭灰色背景（#0b0e11) 以动态青色 (#99f7ff），赛博紫（#bf81ff），以及霓虹绿（#75ffb2）。
3. 玻璃态：所有面板均使用背景模糊 XL采用微妙的原色边框，营造出层次感和高级感。

## 技术细节：
1. 框架：React 18 与 Vite 和 TypeScript。
2. 造型：Tailwind CSS v4，支持自定义主题变量。
3. 动画： 运动实现视图和交互元素之间的平滑过渡。
4. 图表： 重新绘制图表用于响应迅速、高性能的数据可视化。
5. 图标： lucide-react用于一套统一、现代的图标集。
